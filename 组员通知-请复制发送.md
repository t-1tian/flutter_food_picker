1️⃣ 第一次拉取代码（所有人必做）
打开 PowerShell / 终端：
git clone https://github.com/t-1tian/flutter_food_picker.git
cd flutter_food_picker

2️⃣ 切到自己的分支（用你的角色名替换 X）
git checkout -b dev/X
git push -u origin dev/X

   A 同学 → dev/A
   B 同学 → dev/B
   C 同学 → dev/C
   D 同学 → dev/D
   E 同学 → dev/E
   F 同学 → dev/F

3️⃣ 以后提交代码的固定流程
git add .
git commit -m "feat: 简明说明这次改了什么"
git push origin dev/X

4️⃣ 合并到 main（不要直接 push main！）
   ① 在 GitHub 仓库页面点「Compare & pull request」
   ② base 选 main，compare 选 dev/X
   ③ 写清楚改了啥 @F 同学
   ④ F 同学 review 通过后合并

5️⃣ 推送前自检
   ✅ flutter run 编译通过
   ✅ 关键功能能跑
   ✅ 截图保存到 screenshots/ 文件夹
   ✅ 截图一起 commit

6️⃣ 想看效果但本地没装 Flutter？
   👉 https://t-1tian.github.io/flutter_food_picker/
   F 同学合并代码后会自动重新部署

⚠️ 注意事项
• 永远在自己分支上工作，别动 main
• 每天开工前先 git pull origin main 拉最新
• 不会用 git 截图发群里，F 同学一对一教
• 有 BUG 在 GitHub Issues 里提
