# Member E 交付文档 — 使用说明页 & 隐私合规页

## 概述

成员 E 负责应用的两个辅助信息页面：**使用说明页（GuidePage）**和**隐私合规页（PrivacyPage）**，以及独立的**隐私合规性说明文档**。这两个页面为用户提供了完整的使用指引和合规声明，满足课程对项目规范性的要求。

---

## 一、使用说明页（GuidePage）

### 1.1 页面功能

使用说明页以卡片式布局展示应用的操作指南，帮助新用户快速上手。

**页面结构：**

| 区域 | 内容 |
|------|------|
| 顶部标题卡 | 图标 + 「今天吃啥 · 使用教程」标题 |
| 功能介绍 | 应用核心功能的文字描述 |
| 操作步骤 | 分步骤说明如何使用（打开应用 → 点击抽取 → 查看结果 → 重复或刷新） |
| 注意事项 | 网络要求、离线提示等 |

### 1.2 技术实现

- 使用 `ListView` 实现滚动布局，适配不同屏幕高度
- 卡片组件复用 Material Design 3 的 `Card` + 圆角样式
- AppBar 标题居中对齐，与全局主题风格一致
- 文字内容支持自适应缩放

**文件位置：**`lib/pages/guide_page.dart`

---

## 二、隐私合规页（PrivacyPage）

### 2.1 页面功能

隐私合规页展示项目的数据使用政策和合规声明。

**页面结构：**

| 区域 | 内容 |
|------|------|
| 顶部标题 | 「隐私合规」+ 副标题 |
| 合规声明 | 项目名称、版本号、负责人信息 |
| 数据收集 | 明确声明「不收集、不存储、不上传」任何个人信息 |
| 权限清单 | 表格列出各项权限的申请状态和用途 |
| 法律法规 | 引用《网络安全法》《个人信息保护法》等 |
| 联系方式 | 教师和组长联系渠道 |

**文件位置：**`lib/pages/privacy_page.dart`

---

## 三、隐私合规性说明文档

独立于代码之外的完整合规文档，存放于 `docs/隐私合规性说明文档.md`。

### 3.1 文档结构

| 章节 | 内容概要 |
|------|----------|
| 一、应用概述 | 项目定位、版本、负责人、日期 |
| 二、数据收集声明 | 不收集用户隐私（7 类信息列举）、不获取地理位置、网络权限说明 |
| 三、餐饮数据说明 | 数据来源（自行整理编辑）、格式（JSON）、性质（公开测试数据） |
| 四、法律法规合规 | 引用 4 部法律/规范 |
| 五、权限清单 | INTERNET 等 9 项权限逐一说明申请状态 |
| 六、联系方式 | 课程教师 + 项目组长 |
| 七、版本记录 | v1.0.0 初始版本 |

### 3.2 核心声明要点

> 本应用 **不收集、不存储、不上传** 用户的任何个人信息。
> 应用全程本地运行，随机结果仅在当前设备显示，**不会发送至任何远程服务器**。
> 仅在加载菜品数据时使用网络（GET 请求，不携带任何用户数据）。

### 3.3 引用法律法规

- 《中华人民共和国网络安全法》
- 《中华人民共和国个人信息保护法》
- 《App 违法违规收集使用个人信息行为认定方法》
- 《常见类型移动互联网应用程序必要个人信息范围规定》
